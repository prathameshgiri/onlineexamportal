# 📌 1. Project Title
# 🎓 Online Examination System

## 📖 2. Description
This is a full-stack Online Examination Systems web application where users can browse topics, take timed exams, and track their scores, while admins can manage users, view test statistics remotely, and generate real-time analytics.

## 🚀 3. Features
- User Authentication (Login/Register with JWT)
- Auto-fill functionality for Demo tests
- Dynamic Examination Engine with 20-minute constraints
- Order Tracking System (Exam Tracking & Results History)
- Admin Dashboard (Isolated on Port 3001)
- Responsive UI with premium Glassmorphism animations

## 🛠️ 4. Tech Stack
**Frontend:** HTML5, CSS3, JavaScript (Vanilla ES6)
**Backend:** Node.js, Express.js
**Database:** JSON-based persistent local storage
**Other:** APIs, Authentication System (Bcrypt, JWT)

## ⚙️ 6. Installation & Setup
### Step-by-step instructions:
1. Clone the repository
2. Run `npm install`
3. Generate users: `node seed-users.js`
4. Start server: `node server.js`
5. Open `http://localhost:3000` (User) & `http://localhost:3001/admin` (Admin)
